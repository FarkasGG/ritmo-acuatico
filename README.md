# Ritmo Acuático

Proyecto escolar de diseño y desarrollo web realizado con **React** aplicando la metodología **Scrum**.

## Descripción
Ritmo Acuático es un prototipo de tienda en línea orientada a productos acuáticos.
El proyecto presenta una primera versión funcional enfocada en la estructura visual,
identidad gráfica y un catálogo de productos.

## Metodología
Se aplicó la metodología **Scrum**, utilizando:
- Product Backlog
- Sprint Backlog
- Sprint 1 completamente ejecutado

La planeación y seguimiento del sprint se realizó mediante Trello.

## Sprint 1 – Alcance
Durante el Sprint 1 se desarrollaron las siguientes historias de usuario:
- Landing page del sitio
- Identidad visual básica
- Catálogo de productos con imágenes reales
- Diseño responsive (móvil, tablet y escritorio)

## Tecnologías utilizadas
